# App Development 2 – Final Project

Full-stack e-commerce app  
**Backend:** Spring Boot (8080) • **Frontend:** Angular (4200) • **DB:** MySQL

## How to Run
**Backend**
cd ecommerce
mvn spring-boot:run

**Frontend**
cd exercise
ng serve

**DB**
Import SQL files from `/ecom` into MySQL.

## Tech Stack
Java 17 • Spring Boot • Angular • MySQL • Maven • Node/NPM
